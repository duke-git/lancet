## Documentation:

### <span id="Tag">Tag</span>

<p>Get a `Tag` of the `Field`, `Tag` is a abstract struct field tag</p>

<b>Signature:</b>

```go
func (f *Field) Tag() *Tag
```

<b>Example:</b>

```go
package main

import (
	"fmt"
	"github.com/duke-git/lancet/v2/structs"
)

func main() {
	type Parent struct {
		Name string `json:"name,omitempty"`
	}
	p1 := &Parent{"111"}

	s := structs.New(p1)
	n, _ := s.Field("Name")
	tag := n.Tag()

	fmt.Println(tag.Name)
	
	// Output:
	// name
}
```

### <span id="IsEmbedded">IsEmbedded</span>

<p>Check if the field is an embedded field</p>

<b>Signature:</b>

```go
func (f *Field) IsEmbedded() bool
```

<b>Example:</b>

```go
package main

import (
	"fmt"
	"github.com/duke-git/lancet/v2/structs"
)

func main() {
	type Parent struct {
		Name string
	}
	type Child struct {
		Parent
		Age int
	}
	c1 := &Child{}
	c1.Name = "111"
	c1.Age = 11

	s := structs.New(c1)
	n, _ := s.Field("Name")
	a, _ := s.Field("Age")
	
	fmt.Println(n.IsEmbedded())
	fmt.Println(a.IsEmbedded())
	
	// Output: 
	// true
	// false
}
```
